# javaProjects

Small demo project containing a simple program to estimate paint required for a wall.

Build and run
---------------

Compile:

```bash
javac Paint1.java
```

Run (interactive):

```bash
java Paint1
```

Quick test (non-interactive example):

```bash
# provides height=10 and width=12 as example inputs
printf "10\n12\n" | java Paint1
```
